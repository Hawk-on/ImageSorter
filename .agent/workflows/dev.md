# Development Server

Start the Tauri development server with hot-reload for both frontend and backend.

## Prerequisites
- Rust and Cargo installed
- Node.js and npm installed
- Tauri CLI installed

## Steps

// turbo
1. Navigate to project root and start development server:
```bash
npm run tauri dev
```

This will:
- Start the Vite dev server for the frontend
- Compile the Rust backend
- Open the application window with hot-reload enabled
